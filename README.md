# UserManagementAPI

This project is a simple User Management API built with ASP.NET Core Web API.

## Features
- CRUD operations for managing users
- Data validation using Data Annotations
- Custom logging middleware
- RESTful API design

## Tools & Technologies
- ASP.NET Core Web API
- C#
- GitHub
- Microsoft Copilot

## Copilot Usage
Microsoft Copilot was used to scaffold the project, generate CRUD endpoints, and assist with debugging and refactoring the code.

## Endpoints
- GET /api/users
- GET /api/users/{id}
- POST /api/users
- PUT /api/users/{id}
- DELETE /api/users/{id}
